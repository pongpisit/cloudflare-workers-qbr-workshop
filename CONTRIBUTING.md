# Contributing to Cloudflare Workers Dashboard Workshop

Thank you for your interest in contributing! This workshop is designed to be beginner-friendly and accessible.

---

## How to Contribute

### Report Issues

Found a bug or unclear instruction?

1. Go to [GitHub Issues](https://github.com/your-username/cloudflare-workers-dashboard-workshop/issues)
2. Click **New Issue**
3. Describe the problem clearly
4. Include screenshots if helpful

### Suggest Improvements

Have an idea to make the workshop better?

1. Open an issue with the label `enhancement`
2. Describe your suggestion
3. Explain why it would help learners

### Submit Changes

Want to fix something or add content?

1. Fork the repository
2. Create a branch: `git checkout -b fix/your-fix-name`
3. Make your changes
4. Commit: `git commit -m "Fix: description of change"`
5. Push: `git push origin fix/your-fix-name`
6. Open a Pull Request

---

## Guidelines

### Writing Style

- Keep language simple and beginner-friendly
- Use clear, short sentences
- Avoid technical jargon without explanation
- Include examples and screenshots

### Code Examples

- Test all code before submitting
- Include comments explaining what the code does
- Use consistent formatting
- Ensure code works in the Cloudflare Dashboard

### Documentation

- Use clear headings and structure
- Include step-by-step instructions
- Add troubleshooting sections
- Link to official Cloudflare docs

## Areas for Contribution

### Content Improvements
- Clarify confusing sections

### New Modules
- Advanced Workers features
- Integration with external APIs

### Translations
- Translate modules to other languages
- Adapt examples for different regions
