# Cloudflare Workers Dashboard Workshop

Learn to build applications with **Cloudflare Workers** using the **Cloudflare Dashboard**. Complete in **2 hours**.


---

## What You Will Learn

- Navigate the Cloudflare Dashboard
- Create Workers without using the CLI
- Store assets in R2 and data in D1
- Build an Instagram-style photo feed
- Add AI chat + image generation

---

## Workshop Modules

| # | Module | Description |
|---|--------|-------------|
| 1 | [Dashboard Setup](./docs/01-dashboard-setup.md) | Create Cloudflare account, explore the Dashboard |
| 2 | [Hello World Worker](./docs/02-hello-world-worker.md) | Create first Worker, return HTML/JSON |
| 3 | [Profile Page](./docs/03-profile-page.md) | Build personal profile page with styling |
| 4 | [R2 Storage Setup](./docs/04-r2-storage.md) | Create R2 bucket, upload images |
| 5 | [D1 Database Setup](./docs/05-d1-database.md) | Create D1 database, add captions |
| 6 | [Photo Gallery App](./docs/06-photo-gallery.md) | Build Instagram-style app with R2 + D1 |
| 7 | [Workers AI Integration](./docs/07-workers-ai.md) | Add AI chatbot + image studio |

## What You Need

- Windows or macOS

1. Go to [Module 1: Dashboard Setup](./docs/01-dashboard-setup.md)

---

## What You Will Build

By the end of this workshop, you will have:

1. **Hello World Worker** - Your first serverless function
2. **Personal Profile Page** - A styled profile with links
3. **Image Gallery** - Store and serve images from R2
4. **Photo App** - Upload photos with captions (R2 + D1)
5. **AI Chatbot** - Chat with multiple AI models
6. **Complete Application** - Full-featured app deployed to the internet

---

## Technologies Used

| Technology | What It Does |
|------------|-------------|
| **Cloudflare Workers** | Serverless functions on the edge |
| **Cloudflare R2** | Object storage for images and files |
| **Cloudflare D1** | SQLite database at the edge |
| **Workers AI** | AI models (Llama, Mistral, etc.) |
| **Cloudflare Dashboard** | Web interface for everything |
