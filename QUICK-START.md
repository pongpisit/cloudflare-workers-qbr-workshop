# Quick Start Guide

Get started with the Cloudflare Workers Dashboard Workshop in 5 minutes.

---

## What You Need

- Free Cloudflare account (no credit card)
- Web browser
- Internet connection

---

## Step 1: Create Cloudflare Account (2 minutes)

1. Go to [https://dash.cloudflare.com/sign-up](https://dash.cloudflare.com/sign-up)
2. Enter your email
3. Create a password
4. Verify your email
5. You're in!

---

## Step 2: Start Module 1 (3 minutes)

1. Open [Module 1: Dashboard Setup](./docs/01-dashboard-setup.md)
2. Follow the steps
3. Create your first Worker
4. Deploy it
